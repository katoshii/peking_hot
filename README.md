# peking hot 北京热

## 题记

我们终将逝去，但这场「今天北京有点热」的表演值得留下印记。

它用再创作的方式记录下在当代中国网络环境中你能看到的种种恶毒揣测和混乱逻辑。

或许以后浑水会变清，我们能笑着回顾这幕荒诞剧；或许它会愈发恶臭，那时就让这份记录成为研究材料吧。
